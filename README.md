# 🎡 Pokémon GO 對戰盃賽轉盤 (PvP GO Wheel)

這是一個專為 Pokémon GO 玩家設計的互動式轉盤網頁工具。當你與朋友進行 PvP 對戰，卻猶豫不決該玩哪種盃賽時，這個轉盤可以幫你們隨機決定，並自動顯示該盃賽的詳細限制規則。

## 🌟 功能特色

- **互動式彩色轉盤**：流暢的 3 秒旋轉動畫與物理減速效果。
- **自動規則解析**：轉盤停止後，自動彈出詳細的對戰限制，包含：
  - **CP 限制**：500、1500、2500 或無上限。
  - **屬性限制**：特定屬性或禁用的寶可夢資訊。
  - **特殊規範**：是否允許使用「傳說/幻之寶可夢」或「Mega 進化」。
- **慶祝特效**：決定盃賽後會有五彩碎紙 (Confetti) 效果，增添對戰儀式感。
- **響應式設計**：支援手機與電腦瀏覽器開啟。

## 🏆 內建盃賽清單

目前已收錄多種經典與特殊規則，包含：
- **三大聯賽**：超級、高級、大師聯賽。
- **特殊規則**：進化盃、元素盃、萬聖節盃、化石盃。
- **最新加入**：小小盃 (CP 500)、假日盃 (6種屬性)、掛軸盃 (水/格/惡，禁西獅)。

## 🚀 如何使用

1. **線上使用**：
   直接點擊 [你的 Streamlit 網址](https://你的網址.streamlit.app/) 即可開始旋轉。

2. **本地執行**：
   如果你想在自己電腦上執行，請確保已安裝 Python，並執行：
   ```bash
   pip install streamlit
   streamlit run PvPokemon_wheel.py
